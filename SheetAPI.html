Class: SheetAPI

SheetAPI(config)

SheetAPI - Google Sheets Database Wrapper A comprehensive JavaScript library that transforms Google Sheets into a powerful, REST-style database backend. Provides full CRUD operations, advanced querying, intelligent caching, and automatic type casting.

Example
// Basic usage
const api = new SheetAPI({
  spreadsheetId: "your-spreadsheet-id",
  accessToken: "your-oauth-token",
  sheetName: "Users"
});

// Initialize and use
await api.init();

// Create a record
const result = await api.create({ name: "John", age: 30 });

// Get all records
const users = await api.getAll();

// Find by ID
const user = await api.getById("user-123");
